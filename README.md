# resourcepack-autobuilder

Автосборщик ресурс-пака на Python: пакетная генерация конфигурационных файлов из параметров, инкрементальные обновления по хэшам и автоматическая публикация в Git-репозиторий. Написан с нуля.

## Возможности

- Пакетная генерация конфигов (`model.json`, `model.properties`) из набора параметров в цикле
- Инкрементальные обновления: пересобираются и выгружаются только изменённые файлы (сравнение по хэшам)
- Автоматическая публикация результата в Git-репозиторий
- Автономная работа: запуск без участия человека

## Стек

Python · автоматизация Git (`subprocess`) · хэширование · работа с файловой системой и JSON.

## Как устроено

1. Читает параметры моделей из исходных данных.
2. Для каждого элемента генерирует конфигурационные файлы и записывает их в структуру пака.
3. Считает хэши, определяет изменившиеся файлы и собирает только то, что обновилось.
4. Коммитит и пушит результат в репозиторий-витрину, откуда его забирают пользователи.

## Запуск

```bash
git clone https://github.com/1NFERR/resourcepack-autobuilder
cd resourcepack-autobuilder
pip install -r requirements.txt
cp .env.example .env   # впишите свои значения
python build.py
```

## Конфигурация

Секреты вынесены в переменные окружения (файл `.env`, в репозиторий не коммитится):

- `GITHUB_TOKEN` — токен для автопубликации
- `GITHUB_REPO` — репозиторий назначения
- `GITHUB_BRANCH` — ветка назначения

Пример — в `.env.example`.
